### About
This is a case study for my UET.CN3124 class. A **binary exploitation** type CTF in **LA CTF 2023** named **redact**.

### Purpose
Other than to understand the basics of binary exploitation, since I've already working with C++ and later will have to deal with `gdb`, 
I think **pwn** is a greate choice to pracetice my foundational `gdb` skills. This is the chance to escape from _the cave_ lol.

### Setup & Usage
Involve `docker` for repeatability and `Make` for ease of debugging.

#### Build the binary
```
make
```

#### Run the binary
```
make run
```

#### Debug with gdb in run image 
```
make debug
```
